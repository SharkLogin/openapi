### 基礎資訊

> GET /status

介面描述：用於檢查API介面的可用性。

### 請求示例
```
http://localhost:50213/status
```

### 返回數據

| 參數名稱 | 類型   | 說明  |
| :------- | :----- | :------ |
| code     | int    | 執行狀態碼 0 成功 -1 輸入格式錯誤 -2 獲取數量異常 -4 帳號登錄異常 |
| msg      | string | 返回成功或者失敗消息                                           |


執行成功

```json
{
  "code": 0,
  "msg": 'success',
}
```
