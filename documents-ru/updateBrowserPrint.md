### Базовая информация

> POST /api/v2/userapi/user/update

### Параметры запроса

```json
{
  "browser": []
}
```

`browser`  конфигурация массива：

| Имя параметра | Тип | Обязательно | Примечания                    |
| :--------- | :------ | :---- | :--------------------------------- |
| name      | string | Нет | Имя браузера отпечатков пальцев |
| browserid | string | Да | Изменить идентификатор окна браузера |
| proxy    | object | Нет | Информация о прокси - сервере (подробнее см. конфигурацию "proxy") |
| accounts | object | Нет | Информация о номере учетной записи (подробнее см. конфигурацию "accounts") |
| finger   | object | Нет | Информация об отпечатках пальцев браузера (подробнее см. конфигурацию "finger") |

`proxy`  конфигурация структуры：

| Имя параметра | Тип | Обязательно | Примечания                                                                                                                                                         |
| :-------- | :----- | :--- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| type      | string | Да | Класс прокси имеет следующие опции: official (список прокси - серверов платформы), self (собственный список прокси - серверов), socks5 (теперь добавлен), http (теперь добавлен), https (теперь добавлен), zima (теперь добавлен), local (локально) |
| uuid      | string | Нет | Номер прокси - сервера Когда type является официальным или self, заполните эту опцию, чтобы идентифицировать уникальный идентификатор прокси - сервера, который можно просмотреть в свободном прокси - ящике платформы интерфейса UI |
| public_ip | string | Нет | Ваш прокси - сервер экспортирует IP для определения местоположения используемого IP - адреса, если IP - адресация не может быть выполнена для настройки профиля браузера |
| socks5    | object | Нет | Настройка прокси при настройке type для socks5                                                      |
| http      | object | Нет | Настройка прокси при настройке type для http                                                             |
| https     | object | Нет | Настройка прокси при настройке type для https                                                         |
| product     | int | Нет | Настройка прокси при настройке type как official                                                      |

`socks5`  конфигурация структуры：

| Addr          | string | Да          | Адрес агента     |
| User          | string | Нет         | Имя пользователя |
| Passwd        | string | Нет         | Пароль           |

`accounts `  конфигурация структуры：

| Имя параметра | Тип      | Обязательно | Примечания                                                   |
| :------------ | :------- | :---------- | :----------------------------------------------------------- |
| openurls      | []string | Нет         | Обратите внимание, что URL - адрес, который вы посещаете при открытии браузера, в настоящее время поддерживается только 1 добавление. |
| groupid       | string   | Нет         | Кластерный идентификатор создает новую среду отпечатков пальцев, сгруппированную в указанное место |
| user          | string   | Нет         | Аккаунт платформы для автоматического входа в учетную запись среды URL |
| passwd        | string   | Нет         | Пароль учетной записи платформы для автоматического входа в URL |
| cookie        | string   | Нет         | Номер учетной записи cookie или файл cookie для открытия страницы не может быть заполнен одновременно |

`finger ` конфигурация структуры：

| Имя параметра  | Тип     | Обязательно | Примечания                                               |
| :------------- | :------- | :--- | :----------------------------------------------------------- |
| UaVersion      | int      | Нет | Большая версия браузера в настоящее время поддерживает 100 - 117 без передачи автоматической генерации |
| UserAgent      | string   | Нет | UserAgent Не записывать автоматически в соответствии с версиями системы и браузера |
| Language       | []string | Нет | Язык браузера автоматически генерирует подробный список поддерживаемых языков на основе IP - адреса прокси - сервера (если динамический IP автоматически рождается на китайском языке) |
| Zone           | string   | Нет | Список поддерживаемых часовых поясов автоматически генерируется на основе IP - адреса прокси - сервера (если используется динамический IP - адрес для автоматической генерации времени в Пекине) |
| DPI            | string   | Нет | Автогенерация разрешения экрана       |
| FontList       | []string | Нет | Список шрифтов не передается автоматически |
| WebRTC         | int      | Нет | Chrome Компонент мгновенной связи, поддержка: 3 Конфиденциальность;  2 Замена, использование прокси - IP для покрытия реального IP, использование прокси - сцены;  1 Реальный, сайт получает реальный IP;  4 Отключить (по умолчанию), сайт не получит IP |
| WebRTCIP       | string   | Нет | Интранет ip;  Настройка WebRTC для 2 часов |
| Canvas         | int      | Нет | Браузер Canvas Отпечатки пальцев переключатель 1 Stealth 2 Тенденции 3 Тенденции Согласованность |
| WebGl          | int      | Нет | Браузер webgl метаданные переключатель отпечатков пальцев 1 Stealth 2 Реальный (по умолчанию) |
| AudioContext   | int      | Нет | 1 Скрытый 2 Реальный                              |
| SpeechVoices   | int      | Нет | Отпечатки пальцев SpeechVoices, 1: Каждый браузер использует SpeechVoices 2 по умолчанию для текущего компьютера: Добавляет соответствующий шум и создает разные SpeechVoices для каждого браузера на одном и том же компьютере (по умолчанию) |
| MediaDevice    | int      | Нет | Переключатель мультимедийного устройства, 1: Закрыть (каждый браузер использует идентификатор мультимедийного устройства по умолчанию для текущего компьютера) 2: Включить (используйте соответствующие значения вместо вашего реального идентификатора мультимедийного устройства, шум) (по умолчанию) |
| Cpu            | int      | Нет | Количество ядра ЦП не передается автоматически. |
| Mem            | float64  | Нет | Параметры памяти не передаются автоматически. |
| DeviceName     | string   | Нет | Имя компьютера автоматически генерируется без передачи (в пределах 15 символов) |
| Mac            | string   | Нет | MAC - адрес не передается автоматически. |
| Hardware       | int      | Нет | Аппаратное ускорение 1 Открыть (по умолчанию) 2 Закрыть |
| Bluetooth      | int      | Нет | Bluetooth 1 Открыть 2 Закрыть (по умолчанию) |
| DoNotTrack     | int      | Нет | Не отслеживать "Параметры браузера 1 Не включено 2 Включено 3 по умолчанию (по умолчанию) |
| EnableScanPort | int      | Нет | Защита от сканирования порта 1 Открыть (по умолчанию) 2 Закрыть |
| ScanPort       | []int    | Нет | Белый список 0 ~ 65535 Закрытый статус Не записывается Когда EnableScanPort 1 здесь автоматически генерируется локальный порт для пустого |
| geographic     | object   | Нет | Географическое расположение (по умолчанию использование динамического прокси - адреса IP для определения местоположения не поддерживается как запрещенный)） |

`geographic`  конфигурация структуры：

| Имя параметра | Тип | Обязательно | Примечания                       |
| :-------- | :------ | :--- | :----------------------------------- |
| enable    | int     | Нет | Включить 1 (по умолчанию) Запрос 2 Запретить 3 |
| useip     | int     | Нет | 1 Использовать IP - позиционирование (по умолчанию), 2 Использовать пользовательские настройки |
| longitude | string  | Нет | Длительность, если enable равен 1 или 2 и UseIP равен 0 |
| latitude  | string  | Нет | Широта                           |
| accuracy  | string  | Нет | Точность (м) |

Вернуться к данным

```json
{
    "code":0,
    "msg":"",
    "datas":{
        "browse":[
            {
                "id":"fdrgdgsfgsertgreaay345754555",
                "name":"Fingerprint browser",
                "code":0
            },
        ]
    }
}
```

| Имя параметра | Тип | Примечания                        |
| -------- | ------ | :------------------------------------- |
| code     | int    | 0 Успех - 1 Ошибка формата ввода - 4 Ошибка входа в учетную запись |
| msg      | string | Возвращение сообщений об успехе или неудаче |

`browse` Структурный массив：

| Имя параметра | Тип | Примечания               |
| -------- | ------ | :---------------------------- |
| name     | string | Автобиография интерфейса имен окон |
| id       | string | Окно отпечатков пальцев ID |
| code     | int    | 0: Успех, - 1 Не найден соответствующий браузер |
